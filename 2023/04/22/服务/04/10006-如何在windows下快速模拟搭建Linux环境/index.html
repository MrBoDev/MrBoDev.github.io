准备工作

  • 下载安装 Oracle VM VirtualBox

  • 下载安装 vargent

  • 下载virtualbox镜像 centos7或centos8

  • 下载基础配置 Vagrantfile-centos7或Vagrantfile-centos8

安装Centos7操作系统

进入我们事先创建好的目录 vagrant

创建一个目录存放接下来要安装的centos7操作系统

如 bogedev-centos7-1

在该目录下执行cmd命令,切入到命令行

执行vagrant init命令完成系统配置初始化

查看目录下是否多了一个文件

将事先准备的配置文件内容替换到刚刚生成的配置文件中

配置文件说明

  • config.vm.box = “centos/7”

安装的操作系统镜像 这里指定为事先安装好的本地镜像即可

  • 操作系统的基础配置 内存 cpu 名称等

config.vm.provider “virtualbox” do |vb|
vb.memory = “3000”
vb.name= “boge-centos7-1”
vb.cpus= 2
end
