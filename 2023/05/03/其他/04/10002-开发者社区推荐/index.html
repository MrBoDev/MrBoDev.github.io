16个帮助开发者成长的技术社区

原创 Code So Easy 面向Offer学编程 2021-02-20 13:11

1.StackOverflow
网址：https://stackoverflow.com/
Stack Overflow是最受程序员欢迎的IT技术问答网站，而且也是内容最丰富的社区之一。即时不常用，每次google搜索问题时，也经常链接到此网站。
如果遇到技术问题，google之外，直接来此搜索，搜索不到满意的答案时再自己发起提问。当然，逛这个网站需要不错的英语阅读能力。

2.GitHub
网址：https://github.com/
Github是最活跃的开源代码库和版本控制平台，可以说是程序员当中知名度最高的技术社区。各个领域的开源类库、代码，应有尽有。

3.Reddit
网址：https://www.reddit.com/r/prog…
Reddit是欧美一个非常有意思的在线社区，由数量众多的子版块组成。你可以发起活动，或提交自己感兴趣的话题。和国内的百度贴吧有些类似。
编程板块只是其中的一个板块，在这里可以和世界各地的程序员探讨和交流技术问题或IT热点资讯。

4.Hacker News
网址：https://news.ycombinator.com/
Hacker News 是一家关于计算机黑客和创业公司的社会化新闻网站，由保罗·格雷厄姆的创业孵化器 Y Combinator 创建。与其它社会化新闻网站不同的是 Hacker News 没有踩或反对一条提交新闻的选项（不过评论还是可以被有足够 Karma 的用户投反对票）；只可以赞或是完全不投票。简而言之，Hacker News 允许提交任何可以被理解为“任何满足人们求知欲”的新闻。

5.InfoQ
网址：https://www.infoq.com/
InfoQ是一个面向IT领域的，也是非常有影响力的技术媒体网站。通过提供相对有深度的新闻、文章、演讲，以促进IT领域知识的传播和分享。
InfoQ比较适合中级以上的程序员，架构师等。InfoQ还成立了专门的中文站（网址：http://www.infoq.com/cn/）、日文站（网址：https://www.infoq.com/jp/）等海外版的站点。

6.CSDN
网址：https://www.csdn.net/
CSDN 是中国专业的IT社区，为中国的软件开发者提供知识传播、在线学习、职业发展等全生命周期服务。截止2018年6月，CSDN 拥有超过2500+万技术会员，论坛发帖数1000万+，技术资源700万+，Blog文章1300万+，新媒体矩阵粉丝数量430万+。

7.SegmentFault
网址：https://segmentfault.com/
SegmentFault 创立于 2012 年，是中文领域最大的技术问答交流社区平台，在这里你可以检索，交流和分享任何技术编程相关的问题及知识。SegmentFault 为编程爱好者提供一个纯粹、高质的技术交流的平台， 与开发者一起学习、交流与成长，创造属于开发者的时代。

8.掘金
网址：https://juejin.im/
掘金是一个帮助开发者成长的社区，是给开发者用的 Hacker News，给设计师用的 Designer News，和给产品经理用的 Medium。掘金的技术文章由稀土上聚集的技术大牛和极客共同编辑为你筛选出最优质的干货，其中包括：Android、iOS、前端、后端等方面的内容。用户每天都可以在这里找到技术世界的头条内容。与此同时，掘金内还有沸点、掘金翻译计划、线下活动、专栏文章等内容。即使你是 GitHub、StackOverflow、开源中国的用户，相信你也可以在这里有所收获。

9.开发者头条
网址：https://toutiao.io/
开发者头条是由一群程序员创建的，每天分享各类技术文章，已覆盖百万程序员；
开发者头条旗下包含：独家号和团队号
独家号是开发者个人、媒体、企业或其他组织在开发者头条上申请的应用账号，旨在帮助开发者个人、媒体、企业或其他组织在移动端获得更多曝光和关注，扩大技术品牌影响力。
通过独家号，可在开发者头条上分享链接、吸引粉丝、互动关系。依托开发者头条的用户基数和良好口碑，能够为入驻者创造更大价值。

10.开源中国
网址：https://www.oschina.net/
开源中国可以说是中国版的Github，应该是国内最大的开源技术社区，旗下的“码云”提供非常完善的代码托管、团队协作、质量分析、项目演示等功能。相比Github，毕竟是国内网站，访问速度会比较快，可以作为免费的私有仓库使用。

11.ChinaUnix
网址：http://www.chinaunix.net/
作为一名开发者，应该少不了跟Linux系统打交道。ChinaUnix就是一个以讨论Linux/Unix、服务器、数据库等技术为主的自由、开放的技术社区。如果你平时工作需要操作Linux/Unix，CU还是非常值得一逛的。

12.ITPUB

网址：http://www.itpub.net/

ITPUB也是国内主流的技术交流论坛之一，Oracle、PL/SQL、Mysql等数据库技术是ITPUB的侧重点所在。

13.FreeBuf
网址：http://www.freebuf.com/
FreeBuf是国内活跃度比较高的垂直类网络安全网站。上面有大量国外优秀的技术文章，专业性比较高。比较适合相对资深的极客、黑客人群，以及网站安全类岗位的从业人员。

14.RubyChina
网址：https://ruby-china.org/
Ruby China 是国内最权威的 Ruby 社区，拥有国内所有资深的 Ruby 工程师；由众多爱好者共同维护，致力于构建完善的 Ruby 中文社区。

15.LaravelChina
网址：https://laravel-china.org/
高品质的 Laravel 开发者社区，致力于为 Laravel 和 PHP 开发者提供一个分享创造、结识伙伴、协同互助的论坛。

16.CNode
网址：https://cnodejs.org/
CNode 社区为国内最大最具影响力的 Node.js 开源技术社区，致力于 Node.js 的技术研究。
